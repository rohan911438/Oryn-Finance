================================================================================
                            HOW IT WORKS
================================================================================

CONNECTION FLOW:
1. User clicks "Connect Wallet" button
2. App detects wallets: Fighter (primary), Freighter (fallback)
3. If Fighter installed:
   - Calls fighter.connect()
   - Fighter Wallet popup appears
   - User approves connection
   - App retrieves public address
4. If Fighter not installed:
   - Falls back to Freighter
   - Similar flow as Fighter
5. App fetches balances from Stellar Horizon
6. Displays connected address and balances
7. Stores connection info in localStorage

PERSISTENCE FLOW:
1. User closes/refreshes browser
2. App starts and checks localStorage
3. If previous connection found:
   - Retrieves wallet type (Fighter/Freighter)
   - Checks if wallet is still connected
   - Auto-reconnects silently
   - User sees connected state immediately

BALANCE UPDATE FLOW:
1. Connection established
2. Initial balance fetch from Stellar Horizon
3. Balances displayed in dropdown
4. Every 30 seconds: Auto-refresh balances
5. User can manually refresh via UI

================================================================================
                         USAGE EXAMPLES
================================================================================

CONNECT WALLET:
```typescript
import { useWallet } from '@/contexts/WalletContext';

function MyComponent() {
  const { connect, isConnecting } = useWallet();
  return (
    <button onClick={connect} disabled={isConnecting}>
      Connect Wallet
    </button>
  );
}
```

DISPLAY BALANCES:
```typescript
function BalanceDisplay() {
  const { xlmBalance, usdcBalance, isConnected } = useWallet();
  
  if (!isConnected) return <p>Not connected</p>;
  
  return (
    <div>
      <p>XLM: {xlmBalance}</p>
      <p>USDC: {usdcBalance}</p>
    </div>
  );
}
```

================================================================================
                        TROUBLESHOOTING
================================================================================

ISSUE: "Wallet not installed"
SOLUTION:
  - Install Fighter: https://github.com/stellar-expert/stellar-expert-web
  - Install Freighter: https://freighter.app

ISSUE: "Connection popup doesn't appear"
SOLUTION:
  - Check if wallet extension is enabled
  - Check browser console for errors
  - Disable browser popup blockers for this site
  - Refresh page and try again

ISSUE: "Failed to fetch balances"
SOLUTION:
  - Verify internet connection
  - Ensure testnet account has at least 1 XLM
  - Check that account was funded
  - Try refreshing the page

ISSUE: "Auto-reconnect not working"
SOLUTION:
  - Check if localStorage is enabled
  - Verify wallet extension is installed
  - Check if wallet is still on testnet
  - Check browser console for errors
